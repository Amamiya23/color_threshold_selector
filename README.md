##项目简介
之前使用openmv时，它的ide里有一个阈值选择器，体验我很满意，但是只支持LAB色彩空间，现在我刚好有频繁选取阈值的需求，因此设计了一个基于 Python 和 OpenCV 的颜色阈值选择器。支持多种色彩空间：RGB、LAB、HSV。

##安装依赖
在运行程序之前，请确保安装了所需的库。可以使用以下命令安装：
pip install PyQt5 opencv-python numpy
